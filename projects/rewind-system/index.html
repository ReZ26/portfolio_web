RewindSystemComponent_UNREAL-ENGINE

Role: Gameplay Programmer Tech: Unreal Engine 5.4, Blueprint, Enhanced Input
GitHub Video Build

Overview

A flagship gameplay system that records character state snapshots, rewinds on demand, and resumes from a new state with clear player feedback. The system is designed to be readable, deterministic, and ready for iteration.

  • Demonstrates recording/playback architecture and input-driven state transitions.
  • Ships with VFX, audio, and UI cues to clarify the rewind timing.
  • Organized as a reusable component with documentation and debug hooks.

Key features

  • Snapshot capture for position, rotation, velocity, and animation time with a ring buffer.
  • Playback interpolation to smooth rewind motion without input jitter.
  • Input-gated rewind trigger with Enhanced Input for consistent action mapping.
  • Ghost trail and hologram material applied during rewind for clear feedback.
  • Audio sting and UI hint synchronized with the rewind state change.
  • Expose tuning variables and debug toggles for rapid iteration.

System breakdown

Rewind flow: Recording -> Store snapshots -> Trigger rewind -> Playback/interpolate -> Resume.

History buffer holds fixed-rate snapshots with prune-on-write to keep memory stable.
Enhanced Input action triggers the rewind state change and locks movement inputs while rewinding.
Niagara trail, ghost mesh, and rewind material mark the playback window.

Challenges & solutions

Challenge: Playback felt jittery

Approach: Added interpolation between samples and smoothed velocity blending.

Result: Rewind motion reads as intentional and readable.

Challenge: Input conflicts during rewind

Approach: Input gate + state machine ensures rewind overrides locomotion.

Result: Predictable control and clean transitions.

Challenge: Communicating state change

Approach: Combined UI prompt, audio cue, and ghost VFX.

Result: Players recognize the rewind window instantly.

What I would improve next

  • Persist snapshot metadata for checkpoint-specific rewinds.
  • Expose a designer-facing curve to shape rewind speed.
  • Add network-safe replication for multiplayer experiments.
